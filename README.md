**Compression is an app which:**
- show two types of converssion
- first type is standard, when it converts to sum similar characters
- second type, when it rotates set of data by 90 degrees and converts to sum similar characters

Each method save output file with converting string anf marker at the end of the line- '@' for standard and  '#' for rotates

If you want to start the app, enter 'compression.py --file [file path]'
